🔑 GitHub Token Setup

To trigger automatic data updates from this page, you need a GitHub Personal Access Token with repo + workflow scopes.

Create one at: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
Check repo and workflow, set expiration to 1 year, copy the token.

The token is saved only in your browser (localStorage) and is never sent anywhere except GitHub's API.
